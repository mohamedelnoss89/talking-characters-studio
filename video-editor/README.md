# 🎬 المحرر العربي للفيديو - Studio Pro

برنامج مونتاج فيديو احترافي للديسكتوب بـ Electron بواجهة عربية كاملة (RTL).

---

## 📥 طرق التثبيت

### الطريقة 1: تثبيت كامل (موصى بها للمستخدم العادي) ⭐

1. حمل ملف **`ArabicVideoEditor-Setup-1.0.0.exe`**
2. دبل كليك على الملف
3. اتبع خطوات التثبيت:
   - اختر اللغة (عربي/إنجليزي)
   - اقبل اتفاقية الاستخدام
   - اختر مكان التثبيت (افتراضي: `C:\Program Files\المحرر العربي للفيديو`)
   - اختر إنشاء اختصارات (ديسكتوب + قائمة Start)
4. اضغط **"تثبيت"** ثم **"إنهاء"**
5. سيتم تشغيل البرنامج تلقائياً

✅ **المميزات:**
- أيقونة على الديسكتوب
- أيقونة في قائمة Start
- إزالة تثبيت من Control Panel
- يعمل على كل المستخدمين

---

### الطريقة 2: نسخة محمولة (بدون تثبيت)

1. حمل ملف **`ArabicVideoEditor-Portable-1.0.0.exe`**
2. ضعه في أي مجلد تريده
3. دبل كليك لتشغيله مباشرة

✅ **المميزات:**
- بدون تثبيت
- يعمل من USB
- لا يحتاج صلاحيات Administrator

---

### الطريقة 3: للمطورين (من الكود المصدري)

```bash
# تثبيت الحزم
npm install

# تشغيل في وضع التطوير
npm start

# بناء ملفات التثبيت
npm run build:win          # إنشاء Setup + Portable
npm run build:nsis         # إنشاء Setup فقط
npm run build:portable     # إنشاء Portable فقط
```

**المتطلبات:**
- Node.js 16+
- npm أو yarn
- Windows 10/11 (64-bit)

## 📖 دليل الاستخدام السريع

### 1. استيراد الوسائط
- اضغط زر **"استيراد"** في الأعلى
- أو اضغط على **"+"** في مكتبة الوسائط
- أو اسحب الملفات مباشرة من المستكشف

### 2. إضافة وسائط للتايملاين
- اسحب الوسائط من المكتبة إلى المسار المناسب
- أو اضغط على الوسائط لمعاينتها

### 3. تحرير المقاطع
- **تحريك المقطع**: اضغط واسحب
- **تغيير الحجم**: اسحب من حواف المقطع
- **تقسيم**: اضغط `S` مع تحديد المقطع
- **حذف**: اضغط `Delete` مع تحديد المقطع

### 4. تعديل الخصائص
- اضغط على أي مقطع لعرض خصائصه في اللوحة اليمنى
- عدّل الاسم، البداية، المدة

### 5. تصدير الفيديو
- اضغط زر **"تصدير"** في الأعلى
- اختر الصيغة والجودة وFPS
- اضغط **"تصدير الآن"**

## 🏗️ البنية التقنية

```
video-editor/
├── main.js              # عملية Electron الرئيسية
├── preload.js           # جسر آمن بين Main و Renderer
├── package.json         # إعدادات المشروع والاعتماديات
├── build/
│   ├── icon.ico         # أيقونة البرنامج (للويندوز)
│   ├── icon.png         # أيقونة البرنامج (PNG)
│   └── installer-info.txt
├── src/
│   ├── index.html       # الواجهة الرئيسية (عربي RTL)
│   ├── styles.css       # التصميم (ثيم داكن احترافي)
│   └── renderer.js      # منطق التطبيق الكامل
└── assets/              # الموارد الإضافية
```

### التقنيات المستخدمة
- **Electron** — إطار التطبيقات المكتبية
- **FFmpeg** — معالجة الفيديو الحقيقية
- **fluent-ffmpeg** — واجهة Node.js لـ FFmpeg
- **HTML5/CSS3/JS** — الواجهة الأمامية
- **RTL Support** — دعم كامل للعربية
- **electron-builder** — أداة بناء الـ Installer

## 🔧 للمطورين: كيفية بناء الـ Installer

```bash
# 1. تثبيت الحزم
npm install

# 2. بناء الـ Installer (Setup.exe)
npm run build:nsis

# 3. بناء النسخة المحمولة
npm run build:portable

# 4. بناء الاثنين معاً
npm run build
```

الملفات الناتجة ستكون في مجلد `dist/`:
- `ArabicVideoEditor-Setup-1.0.0.exe` - للتثبيت الكامل
- `ArabicVideoEditor-Portable-1.0.0.exe` - للنسخة المحمولة
